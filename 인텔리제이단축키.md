- 해두면 좋은 단축키
  - split
    - "ctrl + alt + ."
  - split right
    - "ctrl + shift + >"
  - unsplit
    - "ctrl + shift + <"
- 메서드 댑스(hierachy) 보기
  - ctrl + alt + h
- endpoint
  - shift + shift
- statement 단위 코드 이동
  - "alt + shift + up/down"
- 순수 코드줄 단위 이동
  - "ctrl + shift + up/down"
- 메서드 추출
  - ctrl + alt + m
